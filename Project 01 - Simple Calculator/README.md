# Simple Calculator

A simple Java CLI application that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Menu-driven interface
- Handles division by zero
- Continuous execution until exit

## Concepts Used
- Java methods
- Control flow (switch-case)
- Loops
- User input using Scanner
- Basic error handling

## Project Structure
src/
    SimpleCalculator.java

## How to Run
javac -d bin src/SimpleCalculator.java
java -cp bin SimpleCalculator
