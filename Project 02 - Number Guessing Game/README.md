# Number Guessing Game

A simple Java CLI game where the player tries to guess a randomly generated number within limited attempts.

## Features
- Random number generation
- Difficulty levels (Easy, Medium, Hard)
- Limited attempts based on difficulty
- Input validation
- Score calculation based on attempts
- Option to play multiple rounds

## Concepts Used
- Java methods
- Control flow (if-else)
- Loops
- Random number generation
- Input validation using Scanner
- Constants for configuration

## Project Structure
src/
    NumberGuessingGame.java

## How to Run
javac -d bin src/NumberGuessingGame.java
java -cp bin NumberGuessingGame
